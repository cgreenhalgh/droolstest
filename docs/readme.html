Currently status approximately:

  • lobby server interaction sketched out but no lobby service implemented.

  • first example application created (bluetoothex - see above)
  • first version of game server created
    • with first version of client-server protocol (without security)

Installation / deployment

For use from authorapp use ant target "authorapp".

For standalone webapp / container set-up see installation.html.

Deploys by default as context droolstest, e.g. http://localhost:8080/droolstest.
